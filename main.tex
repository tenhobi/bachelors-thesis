\abstractCS{Práce se zabývá návrhem a implementací
kooperativní mobilní hry
pro mobilní operační systémy Android a iOS.
Důraz je kladen na výběr a popis konkurenčních aplikací a technologií.
Na základě rešerše byly vybrány technologie jako
framework Flutter, state management BLoC,
databáze Cloud Firestore a architektura Clean Architecture.
Výsledkem práce je funkční mobilní hra,
ve které je díky dobré vzájemné komunikaci hráčů cílem splnění misí hry.
}
\abstractEN{\todo{upravit překlad}
The thesis describes design and implementation
of cooperative mobile game for Android and iOS mobile operating systems
using the multi-platform Flutter framework.
Emphasis is also placed on the selection and description of
frameworks, state management solutions, databases, architectures
and testing options.
}
